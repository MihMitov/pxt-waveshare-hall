# Waveshare AH49E Hall Sensor for MakeCode

Supports linear Hall-effect sensor AH49E (Waveshare).

## Blocks

- Set pin
- Set sensitivity
- Read field
- Detect magnet
- On magnet detected event
- On magnet removed event

## Wiring

| Sensor | micro:bit |
|-------|----------|
| VCC   | 3V |
| GND   | GND |
| OUT   | P1 (default) |
